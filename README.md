## Overview
This repository documents a real client project: a photography website built for a professional photographer based in Canada.

The goal was to deliver a clean, modern, and easy-to-manage website that highlights portfolio work while remaining fast and mobile-friendly.

## Client
- Professional photographer
- Based in Canada
- Project received via referral

## Tech Stack
- WordPress
- Elementor Pro
- Premium photography theme
- Basic SEO & performance optimization

## My Contributions
- Customized theme layouts and page structure
- Resolved complex header and layout behavior
- Optimized image-heavy pages for performance
- Implemented mobile-first responsive design
- Structured portfolio categories for clarity
- Integrated contact / booking flow

## Challenges & Solutions
**Challenge:** Theme had multiple header systems that conflicted across pages  
**Solution:** Resolved at theme-configuration level instead of Elementor overrides

**Challenge:** Large image galleries affecting load speed  
**Solution:** Image optimization, lazy loading, and responsive sizing
